- targeted research/report tests affected by the branch work
- any catalog or program-report tests touched by evidence-capture changes

## App Checks

- `docker compose ps`
- verify runtime overview, campaign state, and latest artifacts remain readable while the branch runs

- if branch artifacts change the specialist review packs, verify candidate-review and paper-trade-readiness still see the expected compact evidence

- branch evidence stays current in one place
- promotion or retirement can be justified from persisted artifacts
- no policy thresholds are silently weakened
