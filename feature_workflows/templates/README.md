# Workflow Templates

Copy these files into a new workflow folder when planning a new feature or job.

Required file set:

- `00_summary.template.md`
- `01_context.template.md`
- `02_codebase_notes.template.md`
- `03_plan.template.md`
- `04_verification.template.md`
- `05_lessons.template.md`

After copying:

1. rename the files to drop `.template`
2. replace placeholders
3. add the workflow to `../00_program_board.md`
4. add or update any relevant hotspot ownership in `../01_code_hotspots.md`
